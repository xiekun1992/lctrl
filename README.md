# LCTRL_RUST
一个用Rust编写的局域网电脑键鼠控制工具

### 功能列表
- [x] 自动发现
- [x] 受控端自动连接
- [x] 鼠标框选时不进入控制
- [ ] 多屏幕支持
- [ ] 网卡新增/删除检测
- [x] 文件传输
- [ ] Wake On LAN
- [ ] 软件更新
- [ ] 日志输出
- [ ] 拖拽时鼠标不跨屏幕
- [ ] 断连重置键鼠按下操作
- [ ] 控制端快捷键取消控制
- [x] 失活自动解控
- [x] 远端设备存活检测
- [x] Web API
- [x] 手动增加远端设备
- [x] 控制界面
- Windows
1. - [x] 鼠标控制

5. - [ ] 账号登录界面控制
- Linux
1. - [ ] 鼠标控制
